# React Roadmap

Examples of fundamental and advanced topics from [roadmap.sh/react](roadmap.sh/react)

## How to Run

1. Clone repo
2. `cd roadmap` or `cd create-react-app`
3. `npm i` if haven't yet since cloning
4. `npm run start`
5. Open `localhost:3000` in browser
